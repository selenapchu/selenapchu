Titanic: Machine Learning from Disaster

Programming Language: Python
Software: Visual Studio, Jupyter Notebook

Goal of the project

Find insights from the data using pandas to analyze and manipulate the data and matplotlib and seaborn for data visualizations.
Build a predictive model that answers the question: “what sorts of people were more likely to survive?” using passenger data (ie name, age, gender, socio-economic class, etc), applying logistic regression model to the test set and submit the prediction csv file to Kaggle.com. to evaluate the accuracy of the model.

About the dataset

Kaggle's Titanic Machine Learning Competition
Source: Kaggle.com
