Programming Language: SQL
Software: Oracle SQL Developer

DATA EXPLORATION

Data exploration is the first step of data analysis. It is a good way to get to know the dataset before one performs tasks like data visualization.

I was given a bank dataset that consist of 11 tables: ACCOUNT, BRANCH, CUSTOMER, DEPARTMENT, EMPLOYEE, INDIVIDUAL, PRODUCT, PRODUCT_TYPE, BUSINESS, OFFICER, and TRANSACTION, etc. And each table have various attributes. To understand relationships between different tables, I decided to start with SQL queries and use aggregation functions like COUNT(), AVG(), MAX()/MIN(), and other syntaxes such as JOINs and CREATE VIEW to manipulate and extract useful data.
